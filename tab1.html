Why Before It's Forgotten?

Short awnser... I needed to make something for my SIP or my Senior Computer Science Project. AKA a graduation requirment

The longer answer as to why I created BIF is because for my SIP I wanted to somehow combine my CS major and film/ media studies minor together in a cohesive and intriguing way. After some contemplation I knew I had my mind set on attempting to develop a website from scratch and I also wanted to include my very talented and artistic friends and family in the process. Over the last few months there have been many variations of the site you see today but the fundamentals of BIF have always been a website I create that combines a beautiful compilation of the moments, the feelings, and the thoughts that deserve to be remembered.


BIF in Detail

Short awnser... It was a ten-ish week project that uses Bootstrap (an HTML and Javascript framework) with CSS

I had a little bit more than 10 weeks to plan, design, develop, and revise this site. During the planning stage it took me a few days to really decide on a framework to use. I was interested in bootstrap but I also looked into others like ionic, react, etc. In the end I found bootstrap to be the quickest and easiest to pick up- so I built this site using bootstrap, a javascript, html framework with CSS. The web hosting and domain is from godaddy. This website has six pages, one home page, one about page, and four content pages. The main UI components I used were, containers, carousels, modals, infinite scroll,navbar, pagination, and buttons.
